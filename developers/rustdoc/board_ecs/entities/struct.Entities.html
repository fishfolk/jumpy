Expand description

Holds a list of alive entities.

It also holds a list of entities that were recently killed, which allows to remove components of deleted entities at the end of a game frame.

Fields

alive: BitSetVec

Bitset containing all living entities

generation: Vec<u32>killed: Vec<Entity>next_id: usizehas_deleted: bool

helps to know if we should directly append after next_id or if we should look through the bitset.
